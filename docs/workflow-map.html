Portfolio Intelligence Dashboard

Foundation Workflow Map
Folahan Williams · Funds Management & Technology · Thursday 18 June · Week 1
The frontend never computes fund math. The backend never handles presentation.

End-to-end flow

1 — DATA (mock files)
holdings · prices · benchmark · fx   CSV files on disk — no database, no external calls
▼
2 — BACKEND  (Python · FastAPI · pandas/numpy)
Load & validate  →  FX-normalise to USD  →  Compute metrics  →  Serve REST
all fund math lives here · bad rows skipped & flagged · accuracy ±0.1%
▼ JSON over REST  (/summary · /exposure · /risk · /attribution)
3 — FRONTEND  (React · TypeScript · Vite · Tailwind · Recharts)
Fetch JSON  →  Dashboard shell  →  Charts & tables
pure presentation — renders only, holds no fund math
▼
4 — PORTFOLIO MANAGER
One screen: positioning · risk · attribution
change window / filter  ↩  recomputes from backend in < 500 ms
